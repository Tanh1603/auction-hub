# Use Case 3.3.0: Manage Auction

## 1. Use Case Description

| Field              | Content                                                                                                                                       |
| :----------------- | :-------------------------------------------------------------------------------------------------------------------------------------------- |
| **Name**           | Manage Auction                                                                                                                                |
| **Description**    | This use case allows the **Auctioneer** to choose one of the corresponding functions of CRUD to adjust **Auction** information in the system. |
| **Actor**          | **Auctioneer**, **Admin**, **Super Admin**                                                                                                    |
| **Trigger**        | When the **Auctioneer** clicks on the "**Auction**" button on the navigation sidebar.                                                         |
| **Pre-condition**  | 1. **Auctioneer**'s device must be connected to the internet.<br>2. **Auctioneer** is signed in with their account.                           |
| **Post-condition** | The **Auction** information is updated to the corresponding function that the **Auctioneer** executes.                                        |

---

## 2. Sequence Flow

```plantuml
@startuml
autonumber
skinparam sequenceMessageAlign center

actor "Auctioneer" as User
boundary "AuctionManagementPage" as View
control "AuctionController" as Controller
entity "AuctionRepository" as Repo

User -> View: Access Auction Management
View -> Controller: GET /auctions
activate Controller
Controller -> Repo: findAll()
activate Repo
Repo --> Controller: AuctionList
deactivate Repo
Controller --> View: Return list data
deactivate Controller
View --> User: Display list of Auctions

User -> View: Choose function

opt List All Auctions
    ref over View, Controller, Repo: Sequence List All Auctions (3.3.1)
end

opt Get Auction Details
    ref over View, Controller, Repo: Sequence Get Auction Details (3.3.2)
end

opt Create Auction
    ref over View, Controller, Repo: Sequence Create Auction (3.3.3)
end

opt Update Auction
    ref over View, Controller, Repo: Sequence Update Auction (3.3.4)
end

opt Delete Auction
    ref over View, Controller, Repo: Sequence Delete Auction (3.3.5)
end

opt Update Auction Relations
    ref over View, Controller, Repo: Sequence Update Auction Relations (3.3.6)
end

opt Update Auction Resources
    ref over View, Controller, Repo: Sequence Update Auction Resources (3.3.7)
end
@enduml
```
